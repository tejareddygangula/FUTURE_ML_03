# Candidate Hiring Prediction System

This project predicts whether a candidate will be hired based on educational background, technical skills, internships, certifications, and experience using Machine Learning techniques.

---

## Project Objective

The objective of this project is to automate candidate hiring prediction using Machine Learning models and data analysis techniques.

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

## Machine Learning Techniques

- Data Preprocessing
- Feature Engineering
- Random Forest Classification
- Model Evaluation
- Feature Importance Analysis

---

## Features Used

- CGPA
- Internships
- Projects
- Programming Languages
- Certifications
- Experience Years
- Hackathons
- Research Papers
- Skills Score
- Soft Skills Score
- Company Type

---

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Categorical Feature Encoding
4. Train-Test Split
5. Model Training
6. Prediction & Accuracy Evaluation
7. Visualization & Feature Importance Analysis

---

## Results

- Successfully predicted candidate hiring outcomes.
- Identified important hiring factors using feature importance analysis.
- Achieved strong classification performance using Random Forest Classifier.

---

## Business Insights

- Machine Learning can automate recruitment processes.
- Predictive hiring systems reduce HR workload.
- Skills and experience significantly impact hiring decisions.
- AI-based recruitment improves efficiency and decision-making.

---

## Conclusion

This project demonstrates how Machine Learning can assist organizations in predicting candidate hiring decisions using educational and professional attributes.
